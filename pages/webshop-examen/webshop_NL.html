Webshop

Wat is het?

Dit is een webshop waar je een product toe kan voegen en dan laat hij hem zien op de main pagina.

Waarom heb ik dit gemaakt?

Dit project heb ik gemaakt, omdat ik met een database wou gaan werken en dit is simple project waar ik het uit geleerd heb testen.

Waar is het mee gemaakt?

Framework: Vue.js

Databse: mongoDB

Product add pagina

Dit was de pagina waar je producten zelf zou toe kunnen voegen en dan werden ze neer gezet op de home page.

Home pagina

Dit is de home pagina waar je alle producten zou kunnen zien die in de database staan.
